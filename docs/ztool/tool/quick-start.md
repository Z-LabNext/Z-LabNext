# 工具函数

提供了一系列经过实战验证的函数，专为日常业务逻辑设计 —— 从数据格式化、类型判断到异步流程控制，覆盖前端开发高频场景。开箱即用，无需额外配置，助你提升开发效率，专注核心功能实现。

## 安装依赖

::: code-group

```bash [npm]
npm install @zlabnext/ztool
```

```bash [yarn]
yarn add @zlabnext/ztool
```

```bash [pnpm]
pnpm add @zlabnext/ztool
```

:::
